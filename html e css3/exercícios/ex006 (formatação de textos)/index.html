Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito utilizando a tag <b> (não semântica).

Já nesta frase, temos um termo em destaque utilizando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântico).

Já nesta frase, temos um termo com ênfase usando a tag <em> (semântico).

Texto Marcado

Podemos também criar um texto marcado utilizando a tag <mark>.

E no outro parágrafo posso criar outro texto marcado, da cor original ou de outra cor.

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse mesmo parágrafo, utilizando as tags <big> e <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para isso, utilizamos a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso utilizamos a tag <ins>.

Podemos inserir o sublinhado utilizando também a tag <u> (não semântica).

Texto Sobrescrito

Serve para inserir coisas do tipo: x20+3. E utiliza-se a tag <sup>.

Texto Subscrito

Serve para inserir coisas do tipo H20. e Utiliza-se a tag <sub>.
